H1 headers

Headers should be used as the title or subtitle in a given section. The H1 header should only be used for the most important headline/titles on a given page. Any other point of emphasis information should be conveyed using headers h2-h6. The font for an H1 header should be bold, the color black and have a specified font weight of 1000.

Paragraphs

The paragraph should be used to convey messages such as this one. The paragraph should have a background similar to the rest of the page and use the font family Arial, Helvetica, sans-serif.
